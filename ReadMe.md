# Welcome Dali

Compose library to load async images

## Getting started / Prerequisites / Dependencies

```groovy
// build.gradle (:project)
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}

// build.gradle (:app)
dependencies {
    implementation 'com.github.AlexGladkov:Dali:0.0.1'
}
```
